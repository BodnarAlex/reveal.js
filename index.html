What is Canvas

"canvas" is an HTML element which can be used to draw graphics via scripting (usually JavaScript). This can, for instance, be used to draw graphs, combine photos, or create simple animations.

What to use canvas for

  • Game Development
  • Data Visualization
  • Image Editing
  • Drawing Tools

What was used before the canvas

  • SVG (Scalable Vector Graphics)
  • Flash
  • HTML Tables and CSS
  • HTML Image Maps

Drawing shapes

We'll look at drawing the following shapes and more:

  • Rect
  • Line
  • Arcs
  • Curves
  • Text

Rect


						ctx.fillStyle = 'blue';
						ctx.fillRect(50, 50, 20, 30);

Rect


						ctx.strokeStyle = 'red';
						ctx.strokeRect(50, 50, 40, 20);

Line


						ctx.beginPath();
						ctx.moveTo(50, 50);
						ctx.lineTo(80, 60);
						ctx.strokeStyle = 'green';
						ctx.stroke();
						ctx.closePath();

Arcs


						ctx.beginPath();
						ctx.arc(50, 50, 20, 0, Math.PI * 2);
						ctx.strokeStyle = 'violet';
						ctx.stroke();
						ctx.closePath();

Arcs


						ctx.beginPath();
						ctx.arc(50, 50, 20, 0, Math.PI);
						ctx.fillStyle = 'yellow';
						ctx.fill();
						ctx.closePath();

Curves


						ctx.bezierCurveTo(20, 20, 100, 80, 90, 10);
						ctx.strokeStyle = 'braon';
						ctx.stroke();

Text

Colors at Canvas

'strokeStyle' and 'fillStyle' properties

These properties take as a value a string describing the color. This string can be in one of the following formats:

  • 'coral'
  • '#FF7F50'
  • 'rgb(255, 127, 80)'
  • 'rgba(255, 127, 80, 1)'

Transparency

Transparency can be achieved in two ways:

  • using the rgba() function

    
								ctx.fillStyle = "rgba(100,150,185,0.5)";

  • The transparency value can be set using the property

    
								ctx.globalAlpha = 0.333;

Animation

Animation logic

  1. Draw a circle 
    
								ctx.fillStyle = 'blue';
								ctx.arc(50, x, 10, 0, Math.PI * 2);
								ctx.fill();
  2. Erase circle 
    
								ctx.clearRect(0, 0, canvas.width, canvas.height);
  3. Draw a circle lower on the y-axis for the next frame using requestAnimationFrame

animate()


						const canvas = document.getElementById("canvas");
						const ctx = canvas.getContext("2d");
						let x = 0;

					function animate() {
						ctx.clearRect(0, 0, canvas.width, canvas.height);

						x += 1;

						ctx.fillStyle = 'blue';
						ctx.arc(50, x, 10, 0, Math.PI * 2);
						ctx.fill();
						requestAnimationFrame(animate);
					}

					animate();
