What is Canvas

"canvas" is an HTML element which can be used to draw graphics via scripting (usually JavaScript). This can, for instance, be used to draw graphs, combine photos, or create simple animations.

What to use canvas for

  • Game Development
  • Data Visualization
  • Image Editing
  • Drawing Tools

What was used before the canvas

  • SVG (Scalable Vector Graphics)
  • Flash
  • HTML Tables and CSS
  • HTML Image Maps

Colors at Canvas

'strokeStyle' and 'fillStyle' properties

These properties take as a value a string describing the color. This string can be in one of the following formats:

  • 'coral'
  • '#FF7F50'
  • 'rgb(255, 127, 80)'
  • 'rgba(255, 127, 80, 1)'

Transparency

Transparency can be achieved in two ways:

  • using the rgba() function

    
								canvas.fillStyle = "rgba(100,150,185,0.5)";

  • The transparency value can be set using the property

    
								canvas.globalAlpha = 0.333;

Gradient

						var grad = canvas.createRadialGradient(x1, y1, r1, x2, y2, r2)
