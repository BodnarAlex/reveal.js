What is Canvas

"canvas" is an HTML element which can be used to draw graphics via scripting (usually JavaScript). This can, for instance, be used to draw graphs, combine photos, or create simple animations.

What to use canvas for

  • Game Development
  • Data Visualization
  • Image Editing
  • Drawing Tools

What was used before the canvas

  • SVG (Scalable Vector Graphics)
  • Flash
  • HTML Tables and CSS
  • HTML Image Maps

Pros and Cons

It is more convenient to deal with a large number of elements Overloads CPU and RAM
It has hardware acceleration Due to the garbage collector limitation, there is no way to clear memory
Every pixel can be manipulated You need to handle object events yourself
Image processing filters can be applied Poor performance at high resolutions
There are many libraries You have to draw each element separately
